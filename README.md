Este proyecto fue desarrollado para la materia *Laboratorio de Programación* de *6.º año*. Es un sistema de gestión para un gremio de aventureros (RPG) desarrollado en Java utilizando Programación Orientada a Objetos (POO).

#Requisitos Técnicos Cumplidos

1. *Modularidad:* Separación en tres clases: Entidad (`Aventurero.java`), Gestora (`Gremio.java`) y Controlador/Menú (`Main.java`).
2. *Encapsulamiento:* Uso de atributos privados (`private`) con sus respectivos getters y setters.
3. *Lógica avanzada:* Métodos con retorno (`return`) para calcular la fuerza total del gremio y contar héroes caídos.
4. *Menú interactivo:* Implementación de un ciclo `do-while` con estructura `switch`.
5. *Validaciones y manejo de errores:* Uso de `try-catch` para evitar errores por ingresos de datos inválidos (`NumberFormatException`) y validaciones para impedir campos vacíos o números negativos.

#Cómo ejecutar el proyecto

1. Clonar el repositorio.
2. Abrir la carpeta en Visual Studio Code.
3. Asegurarse de tener instalado Java Extension Pack.
4. Ejecutar el archivo `Main.java`.
